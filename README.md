# next-github-oauth
# Next.js => Github OAuth Example

This is an example of how to use Github OAuth with Next.js and Koa.


# How to use

1. Clone the repository
2. Install dependencies: `yarn install`
3. yarn dev
4. Open http://localhost:3000 in your browser
